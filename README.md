# 🎨 Пикассо (от Сбера)

**Пикассо** — это веб-приложение для создания поздравительных открыток прямо в браузере.
Проект реализован на чистом **HTML, CSS и JavaScript** без использования сторонних фреймворков.

---

## 📌 Описание

С помощью редактора пользователь может быстро создать красивую поздравительную открытку:

* выбрать готовый шаблон
* написать текст поздравления
* или полностью кастомизировать открытку под себя

Проект ориентирован на простоту использования и гибкость настройки.

---

## 🚀 Функционал

* 🎨 **Выбор шаблонов фона**
  Готовые стили и дизайны для быстрого старта

* ✍️ **Готовые варианты поздравлений**
  Возможность выбрать или отредактировать текст

* 🧩 **Конструктор открытки**
  Полная кастомизация:

  * Заголовок
  * Обращение
  * Основное описание
  * Подпись

* 🤖 **Генерация промта для ИИ**
  Возможность сформировать текстовый запрос для генерации поздравления с помощью ИИ

## ⚙️ Установка и запуск

1. Склонировать репозиторий:

```

```

2. Открыть файл:

```
index.html
```

Проект не требует сборки или установки зависимостей.

## 💡 Пример использования

1. Выбрать шаблон фона
2. Добавить или выбрать текст поздравления
3. При необходимости — настроить все поля вручную
4. Сгенерировать промт для ИИ (опционально)
5. Сохранить открытку
